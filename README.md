# Sales & Revenue Analysis Dashboard

## Project Overview
The Sales & Revenue Analysis Dashboard is a web-based application that helps users analyze sales and revenue data from CSV files. The dashboard provides key performance indicators (KPIs), data visualization, and an easy-to-use interface for business insights.

## Features
- Upload sales data using CSV files
- Display Total Sales
- Display Total Revenue
- View sales records in a table format
- Simple and interactive dashboard
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Dataset Format

The CSV file should follow the format below:

Date,Product,Sales,Revenue
2025-01-01,Laptop,20,100000
2025-01-02,Mobile,35,70000
2025-01-03,Tablet,15,45000

## How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on index.html.
5. Select "Open with Live Server".
6. Upload the sales.csv file.
7. View the dashboard in your browser.

## Expected Output

The dashboard will display:
- Total Sales KPI
- Total Revenue KPI
- Sales Data Table
